|    函数及使用     |                             描述                             |
| :---------------: | :----------------------------------------------------------: |
|      abs(x)       |                 绝对值, 返回结果为 x的绝对值                 |
|   divmod(x, y)    |        商余, 同时输出商和余数, 返回结果为 (x//y, x%y)        |
|  pow(x, y[, z])   |           幂余, 返回结果为(x**y)%z , z的默认值为1            |
|   round(x[, d])   | 四舍五入, x为要四舍五入的数, d为保留的小数位数, d的默认值为0 |
| max(x1,x2,...,xn) |                          返回最大值                          |
| min(x1,x2,...,xn) |                          返回最小值                          |
