### 比较运算符

| 运算符 | 描述                 |
| ------ | -------------------- |
| ==     | 等于                 |
| ===    | 值相等并且类型相等   |
| !=     | 不相等               |
| !==    | 值不相等或类型不相等 |
| >      | 大于                 |
| <      | 小于                 |
| >=     | 大于或等于           |
| <=     | 小于或等于           |

### 比较不同的类型

在**数字**与**字符串**进行比较时，JavaScript 会把字符串转换为数值。**空字符串**将被转换为 **0**。**非数值字符串**将被转换为始终为 false 的 **NaN**。

当比较**两个字符串**时，"2" 大于 "12"，因为（按照字母排序）1 小于 2。
