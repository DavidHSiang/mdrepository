## 一、安装

ubuntu 下rar解压工具安装方法：

### (1) 压缩功能

```shell
sudo apt-get install rar1
```

### (2) 解压功能

```shell
sudo apt-get install unrar1
```

## 二、使用

1. 可以直接在UI界面使用了
2. `rar x test.rar`
